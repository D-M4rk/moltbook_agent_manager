================================================================================
TROUBLESHOOTING
================================================================================

If you get a syntax error:
1. Make sure you added a COMMA after the closing }
2. Make sure the "Custom" entry is still at the end
3. Check that quotes match correctly

If the personality doesn't appear:
1. Make sure you saved the file
2. Make sure you restarted the app completely
3. Check Python console for any error messages
